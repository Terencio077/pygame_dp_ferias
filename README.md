    Descrição:
              O Futebeças é um jogo de futebol estilo "head ball" desenvolvido em Python utilizando a biblioteca PyGame. O objetivo é controlar personagens cabeçudos para marcar gols no adversário. O jogo conta com física de gravidade, colisões, sistema de pontuação e telas de menu e vitória.
            

    Como roda:
            Ter instalado a biblioteca PyGame
